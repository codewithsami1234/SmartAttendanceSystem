# 📚 Smart Attendance System

A simple Android-based Smart Attendance System developed to manage student attendance efficiently. The application allows users to add students, mark attendance, generate reports, and manage records through an easy-to-use interface.

---

## 🚀 Features

- 🔐 Login System
- 🏠 Dashboard
- 👨‍🎓 Student Management
  - Add Student
  - Edit Student
  - Delete Student
- ✅ Mark Attendance
  - Present
  - Absent
- 📊 Attendance Reports
- ⚙️ Settings
- 🚪 Logout
- 📱 User-Friendly Interface
